〖ミニゲーム作り〗

＜ゲーム説明＞

1~30の中で、ランダムで正解の数字が１つ用意される。
正解が出るまでその数字を探すゲームである。
正解の数字を探し当てることができればゲームクリアとなる。

＜ソースコード＞

  1. import random
  3. def main():
  4.     answer = random.randint(1, 30)
  5.     attempts = 0
  7.     print("1から30までの数字を当ててください。")
  9.     while True:
  10.         try:
  11.             guess = int(input("数字を入力してください: "))
  12.             attempts += 1
  14.             if guess == answer:
  15.                 print(f"正解！{answer}を当てました。{attempts}回目でクリアしました！")
  16.                 break
  17.             else:
  18.                 print("不正解です。もう一度試してみてください。")
  19.         except ValueError:
  20.             print("無効な入力です。整数を入力してください。")
  22. if __name__ == "__main__":
  23.     main()

＜感想＞

時間制限や回数制限、を設けても面白いと思った。
数字を使ったものだとやはりブラックジャックも面白いと思う。
